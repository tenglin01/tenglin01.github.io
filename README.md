# Personal Site (Static)
This is a bilingual (EN/ZH) static site generated locally. Deploy via GitHub Pages.

Quick preview:
- python3 -m http.server -d . 8080
- open http://localhost:8080

Deployment options:
1) User/Org site: <username>.github.io repo, serve from root.
2) Project site: any repo, serve via GitHub Pages workflow to `gh-pages`.
